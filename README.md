# ✋ Advanced Hand Tracking AR

An immersive AI-powered Hand Tracking Augmented Reality experience built using MediaPipe Hands, HTML5 Canvas, CSS3, and JavaScript.

Track your hand movements in real time while generating futuristic particle effects, lightning connections, matrix animations, gesture detection, audio-reactive visuals, and dynamic themes.

## 🚀 How To Use

1. Open the website.
2. Click **Dive In**.
3. Allow camera permissions.
4. Move your hands in front of the camera.
5. Create pinch gestures to trigger shockwaves.
6. Bring both hands closer to generate lightning effects.
7. Switch themes from the bottom control panel.

---

## 🌟 Main Highlights

### ✋ Hand Tracking

Tracks up to two hands simultaneously using MediaPipe Hands.

### ⚡ Lightning Engine

Generates electric arcs between fingertips when hands come close.

### 👌 Gesture Detection

Recognizes pinch gestures and triggers visual shockwaves.

### 🌌 Matrix Background

Animated matrix rain reacts to hand movement speed.

### ✨ Particle System

Creates glowing particles around fingertips.

### 🔊 Audio Engine

Interactive audio responds to hand positions and movement.

### 🎨 Theme Engine

Multiple visual themes with dynamic colors and effects.
